Make the UI feel like a game, not a standard web app. Use motion/animations, playful interactions, big touch targets.

**Why:** The app is for improv theater — it should feel fun and playful, matching the spirit of improv.
**How to apply:** Use a motion library, bouncy animations, large colorful buttons, card-like interactions. Think mobile game UI.
